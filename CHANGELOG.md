# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [2.0.0] - 2024-XX-XX

### Добавлено
- Полный рефакторинг архитектуры проекта
- Система подписок на продукты/версии
- Автоматические уведомления об изменении статуса
- Интеграция с NVD API для поиска CVE
- Мониторинг новых CVE для подписанных продуктов
- Команда `/history` - история запросов пользователя
- Команда `/stats` - статистика для администраторов
- Административные команды (`/admin`)
- Rate limiting для защиты от злоупотреблений
- Персистентный кэш с сохранением на диск
- Retry логика для API запросов
- Улучшенная обработка ошибок
- Inline keyboards для удобного управления
- База данных SQLite с моделями
- Фоновый scheduler для автоматических проверок
- Улучшенный парсер с поддержкой больше форматов
- Fuzzy matching для поиска продуктов
- Структурированное логирование

### Изменено
- Модульная структура проекта
- Улучшенная обработка ошибок
- Оптимизированные запросы к API
- Улучшенное форматирование сообщений

### Исправлено
- Проблемы с обработкой больших списков продуктов
- Ошибки при работе с API
- Проблемы с кэшированием

## [1.0.0] - 2024-XX-XX

### Добавлено
- Базовая проверка версий через endoflife.date API
- Inline режим для использования в любых чатах
- Поддержка файлов (.txt)
- Команды `/start`, `/check`, `/help`
- Docker поддержка
