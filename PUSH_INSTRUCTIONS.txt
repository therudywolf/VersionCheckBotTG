   cd VersionCheckBotTG

3. Убедитесь что вы на main ветке:
   git branch

4. Выполните одну из команд (выберите одну):

   ВАРИАНТ A - Простой push (если GitHub Copilot установлен):
   git push origin main

   ВАРИАНТ B - Push с SSH ключом:
   git push origin main

   ВАРИАНТ C - Push с PAT токеном (если требуется):
   git push https://YOUR_USERNAME:YOUR_PAT@github.com/therudywolf/VersionCheckBotTG.git main

   https://github.com/therudywolf/VersionCheckBotTG/commits/main
